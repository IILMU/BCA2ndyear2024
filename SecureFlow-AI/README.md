# 🚀 SecureFlow AI  
### Intelligent Fraud Detection System

SecureFlow AI is a web-based fraud detection system that analyzes transactions and determines whether they are genuine or suspicious using smart rule-based logic.

---

##  Project Overview
This project simulates how digital payment systems detect fraud.  
It analyzes transaction details and provides:
- ✅ Risk Score
- 📊 Fraud Analysis
- 🔐 Security Alerts

---

##  Features
- 🔍 Transaction Verification  
- 📊 Risk Score (0–100%)  
- ⚡ Real-time Analysis  
- 🧠 Detection based on:
  - Amount  
  - Location  
  - Device  
  - Time  
- 🎯 Clean UI Dashboard  

---

##  Tech Stack
- Python (Flask)  
- HTML, CSS  

---

##  How It Works
Risk score is calculated using rules:

- High Amount → High Risk  
- Unknown/Dubai Location → Suspicious  
- New Device → Risky  
- Night Time → Slight Risk  

###  Result:
- 🟢 Low Risk → Safe  
- 🟡 Medium Risk → Suspicious  
- 🔴 High Risk → Fraud Alert
